 56 ==============================================================================
 57 INTRODUCTION                            vim-efm-langserver-settings-introduction
 58 
 59 vim-efm-langserver-settings is a Vim plugin that auto-enable and work for
 60 efm-langserver (https://github.com/mattn/efm-langserver) under lsp clients.
 61 
 62 This plugin are under development inspired by mattn/vim-lsp-settings
 63 (https://github.com/mattn/vim-lsp-settings).
 64 
 65 Latest version:
 66 https://github.com/tsuyoshicho/vim-efm-langserver-settings.vim
 67 
 68 
 69 ==============================================================================
 70 USAGE                                   vim-efm-langserver-settings-usage
 71 
 72 Installing under plugin-manger usage.
 73 
 74 Now, support lsp clients below:
 75 
 76 - vim-lsp (https://github.com/prabirshrestha/vim-lsp)
 77 - ale (https://github.com/dense-analysis/ale)
 78 - coc.nvim (https://github.com/neoclide/coc.nvim)
 79 
 80 See README for supported commands.

 90                                         efm_langserver_settings#whitelist()
 91 efm_langserver_settings#whitelist()
 92                 Return a list of valid filetype s for running efm-langserver with lsp.
 93                 If included '*' in inner list, return ['*'].
 94 
 95                                 efm_langserver_settings#whitelist_without_any()
 96 efm_langserver_settings#whitelist_without_any()
 97                 Return a list of valid filetype s for running efm-langserver with lsp.
 98                 If included '*' in inner list, return exclude it list.

126 VARIABLES                                 vim-efm_langserver_settings-variables
127 
128 
129 g:efm_langserver_settings#config                g:efm_langserver_settings#config
130     Type: Number
131     Default: 1
132 
133     When set 1(default) to enable config file set to server.
134     When set 0 to disable config file set to server, at this setting use
135     efm-langserver: store config file to default path.
136     (Conversely, without below settings
137      g:efm_langserver_settings#config_file can be reflect custom
138      configuration file)
139 
140 
141 g:efm_langserver_settings#debug                 g:efm_langserver_settings#debug
142     Type: Number
143     Default: 0
144 
145     When set 1 to enable efm-langserver's log-file output.
146 
147 
148                                         g:efm_langserver_settings#config_file
149 g:efm_langserver_settings#config_file
150     Type: String
151     Default: <repo_config>/config.yaml
152 
153     Specify custom efm-langserver config file.
154     Default value is build-in inner config file.
155 
156 
157                                         g:efm_langserver_settings#debug_file
158 g:efm_langserver_settings#debug_file
159     Type: String
160     Default: $HOME/efm-langserver.log
161 
162     Specify efm-langserver debug log file.
163     Default value is generated under $HOME.
164 
165 
166                                 g:efm_langserver_settings#skip_filetype_list
167 g:efm_langserver_settings#skip_filetype_list
168     Type: List
169     Default: []
170 
171      filetype list that is ignored from built-in config settings.
172     But, when using below g:efm_langserver_settings#custom_filetype_list
173     does not work.
174 
175 
176                                 g:efm_langserver_settings#custom_filetype_list
177 g:efm_langserver_settings#custom_filetype_list
178     Type: List
179     Default: []
180 
181     A list to set which filetype of lsp to run efm-langserver when
182     g:efm_langserver_settings#config is set to 0.
