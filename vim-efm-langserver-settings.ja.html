 56 概要                                    vim-efm-langserver-settings-introduction
 57 
 58 vim-efm-langserver-settings は efm-langserver
 59 (https://github.com/mattn/efm-langserver) を lsp クライアント下で自動的に有効
 60 にして活用することを目的としたプラグインです
 61 
 62 このプラグインは mattn/vim-lsp-settings
 63 (https://github.com/mattn/vim-lsp-settings) に触発されて開発中です。
 64 
 65 最新版:
 66 https://github.com/tsuyoshicho/vim-efm-langserver-settings.vim
 67 
 68 
 69 ==============================================================================
 70 使い方                                                                                                                                                                  vim-efm-langserver-settings-usage
 71 
 72 プラグインマネージャの設定に沿ってインストールしてください。
 73 
 74 現在、以下の lsp クライアントと連携して動作します:
 75 
 76 - vim-lsp (https://github.com/prabirshrestha/vim-lsp)
 77 - ale (https://github.com/dense-analysis/ale)
 78 - coc.nvim (https://github.com/neoclide/coc.nvim)
 79 
 80 サポートするコマンドは README を見てください。

130 変数                                      vim-efm_langserver_settings-variables
131 
132 
133 g:efm_langserver_settings#config                g:efm_langserver_settings#config
134     Type: Number
135     Default: 1
136 
137     1を設定(デフォルト)すると efm-langserver の設定ファイルでの内部自動設定を
138     有効にします。
139     0を設定すると efm-langserver の設定を無効にします、efm-langserver を使うな
140     ら efm-langserver がデフォルトで読む設定ファイルを用意する必要があります。
141     (逆に言うと、下記の設定ファイルの指定
142      g:efm_langserver_settings#config_file をしないで自分の設定を反映できます)
143 
144 
145 g:efm_langserver_settings#debug                 g:efm_langserver_settings#debug
146     Type: Number
147     Default: 0
148 
149     1を設定すると efm-langserver のログ出力を有効にします。
150 
151 
152                                         g:efm_langserver_settings#config_file
153 g:efm_langserver_settings#config_file
154     Type: String
155     Default: <repo_config>/config.yaml
156 
157     efm-langserver の設定ファイルを指定できます。
158     初期値はプラグイン同梱のファイルが設定されます。
159 
160 
161                                         g:efm_langserver_settings#debug_file
162 g:efm_langserver_settings#debug_file
163     Type: String
164     Default: $HOME/efm-langserver.log
165 
166     efm-langserver のデバッグログファイルを指定できます。
167     初期値は $HOME 以下に生成するようになっています。
168 
169 
170                                 g:efm_langserver_settings#skip_filetype_list
171 g:efm_langserver_settings#skip_filetype_list
172     Type: List
173     Default: []
174 
175     現在の内部設定から除外する filetype のリスト。
176     ただし、以下の g:efm_langserver_settings#custom_filetype_list が利用され
177     る時には影響しません。
178 
179 
180                                 g:efm_langserver_settings#custom_filetype_list
181 g:efm_langserver_settings#custom_filetype_list
182     Type: List
183     Default: []
184 
185     g:efm_langserver_settings#config を0に設定した際に、efm-langserver を
186     lsp のどの filetype で動作するか設定させるためのリスト。
